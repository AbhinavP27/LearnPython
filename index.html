Variables in Python

A variable is a name used to store data. Python creates variables when you assign values.

Example 1: Store a Value

city = "Kochi"
print(city)
Kochi

Example 2: Update Value

x = 10
x = 20
print(x)
20

Data Types in Python

Data type defines what kind of value the variable holds. Example: number, decimal, text, true/false.

Data Type Example
int 10
float 10.5
complex 2+3j
str "Hello"
bool True
list [1,2,3]
tuple (1,2,3)
set {1,2,3}
dict {"a":1}

int (Integer)

Stores whole numbers (no decimals).

age = 21
print(type(age))
<class 'int'>

float (Decimal)

Stores numbers with decimal points.

price = 99.5
print(type(price))
<class 'float'>

complex (Complex)

Stores numbers with real + imaginary (j).

num = 2+3j
print(type(num))
<class 'complex'>

str (String)

Stores text inside quotes.

name = "Python"
print(type(name))
<class 'str'>

bool (Boolean)

Stores True or False.

is_student = True
print(type(is_student))
<class 'bool'>

list (List)

Stores multiple values (mutable).

nums = [1,2,3]
print(type(nums))
<class 'list'>

tuple (Tuple)

Stores multiple values (immutable).

t = (1,2,3)
print(type(t))
<class 'tuple'>

set (Set)

Stores unique values.

s = {1,2,3}
print(type(s))
<class 'set'>

dict (Dictionary)

Stores data in key-value pairs.

d = {"a":1}
print(type(d))
<class 'dict'>

Type Conversion

Changing one data type into another type.

Implicit Conversion

Python automatically converts type when needed.

a = 5
b = 2.5
c = a + b
print(type(c))
<class 'float'>

Explicit Conversion

You manually convert the type using functions.

x = "10"
y = int(x)
print(type(y))
<class ' int'>
